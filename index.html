Quelle langue

As an avid language learner, I enjoy listening to others speak foreign languages. Sometimes, I'll make a sorry attempt at guessing the language. So I thought it would be interesting if I could create a model that could hopefully do better ...

For this project, I created a model using the Naive Bayes classification algorithm to predict the language of typed text. The data for this project came from Kaggle.
