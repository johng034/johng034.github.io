Quelle langue

As an avid language learner, I enjoy listening to others speak foreign languages. Sometimes, I'll make a sorry attempt at guessing the language. So I thought it would be interesting if I could create a model that ideally could do much better...

For this project, I created a model using the Naive Bayes classification algorithm to predict the language of typed text. The data for this project came from Kaggle.

The Naive Bayes algorithm outperformed the Random Forest and K Nearest Neighbors algorithms. After training the algorithm, I created a web app using Streamlit to allow users to try it out for themselves

Twitter talk

Vocabulary is very important when learning a new language. One of the most effective ways to increase oneâ€™s lexicon is to study the most common words in a language. The goal of this project was to find the most common words used in tweets...

For this project, I used the Twitter API to collect 200 tweets from 10 profiles in 5 different languages. This would result in 10,000 tweets in total (in reality, this number came out lower because some of the accounts did not have 200 tweets on their profile or some tweets were only emojis).

After cleaning the data, I wrote some code to calculate the word frequencies and used the data to create a word cloud for each language. I created a web app where the user can view the top x number of words in both tabular form and as a word cloud for five languages.
